
# AI Interview Copilot

AI Interview Copilot is a **Next.js** application that helps candidates prepare for technical interviews. Upload your resume, generate AI-powered interview questions, provide your answers, and receive detailed feedback on technical skills, communication, and confidence. The app also supports voice input and smart follow-up questions to simulate a real interview experience.  

## Features

- Upload your resume in PDF format  
- Generate tailored AI interview questions based on your projects and skills  
- Type or speak your answers using voice recognition  
- Get AI evaluation scores for **Technical Skills, Communication, and Confidence**  
- Receive AI-generated feedback and follow-up questions  

## Getting Started

Clone the repository and install dependencies:

```bash
npm install
# or
yarn install
# or
pnpm install
````

Start the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) to view the app. Edit the page by modifying `app/page.tsx` — changes reload automatically.

## Technologies Used

* [Next.js](https://nextjs.org) – React framework for server-side rendering and routing
* [Tailwind CSS](https://tailwindcss.com) – Utility-first CSS for styling
* [Recharts](https://recharts.org) – Chart library for visualizing AI evaluation scores
* [FastAPI](https://fastapi.tiangolo.com) – Backend API for handling resume processing and AI requests
* [Google Gemini API](https://ai.google.com) – Generates interview questions and evaluates answers using AI
